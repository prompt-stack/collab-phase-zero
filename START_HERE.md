## 3-Step Deploy

### Step 1: Configure Formspree (3 min)
1. Create free account at https://formspree.io
2. Create new form, copy Form ID
3. Edit `index.html` line 158, paste Form ID

### Step 2: Deploy to GitHub (5 min)
1. Create free GitHub account
2. Create new repository: `collab-mini-site`
3. Push code (see QUICKSTART.md for commands)
4. Enable GitHub Pages in Settings

### Step 3: Create QR Code (2 min)
1. Copy your live URL from GitHub Pages
2. Go to https://www.qr-code-generator.com/
3. Generate & download QR code
4. Test with your phone

**Total Time: 10 minutes**
